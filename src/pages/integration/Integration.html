Le Club FSM_RoboTech a organisé le 13 et 14 October 2018 une formation en programmation Java à la faculté des sciences de Monastir. Le contenu de la formation etait comme le suivant: ✅ La programmation java :écriture d'un premier programme Java, les blocs, les expressions conditionnelles, les boucles ✅ La programmation orientée objet ✅ La programmation avancée Nous remercions les participants pour leur participation enthousiaste.
